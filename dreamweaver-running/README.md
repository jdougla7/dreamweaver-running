# Dreamweaver Running website

This is a deployable React/Vite landing page for Dreamweaver Running.

## Quick start

1. Unzip this folder.
2. Upload the files to a GitHub repository.
3. Import the repository into Vercel.
4. In Vercel, add `dreamweaver.run` and `www.dreamweaver.run` under Project → Settings → Domains.
5. Copy the DNS records Vercel gives you into Namecheap → Domain List → Manage → Advanced DNS.

## Local preview

If you want to preview or edit locally:

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Build

```bash
npm run build
```

## Notes

- Placeholder email is `hello@dreamweaver.run`.
- Main website code is in `src/App.jsx`.
- This version uses gradients and icons, not external image files, so there are no image assets to upload.
