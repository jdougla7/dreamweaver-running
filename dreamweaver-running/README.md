# Dreamweaver Running

Clean Vite + React + Tailwind v4 project configured for Vercel.

## Vercel settings

Framework preset: Vite
Build command: npm run build
Output directory: dist
Root directory: blank, unless this project is inside a subfolder

## Files included

- package.json
- vite.config.js
- index.html
- src/main.jsx
- src/App.jsx
- src/index.css

This project intentionally does not include node_modules. Vercel will install dependencies during deployment.
